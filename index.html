О проекте

Что такое ExcelPro?

ExcelPro — это консольное приложение на языке Python, предназначенное для работы с электронными таблицами. Система предоставляет пользователю полноценный инструментарий для ввода, хранения, обработки и визуализации числовых и текстовых данных.

Архитектура приложения построена по принципу единственной ответственности: каждый из 14 модулей выполняет строго определённую задачу и взаимодействует с другими через чётко определённые интерфейсы.

Проект реализован в рамках учебного курса «Профессиональная практика программной инженерии» студенткой группы ПИ-22б Кущенко Захар Сергеевич.

Архитектура

14 модулей системы

MOD // 01
Пользовательский интерфейс
Рабочая область, ribbon-панель, строка формул, масштабирование и темы оформления.
MOD // 02
Управление файлами
Создание, открытие и сохранение документов. Поддержка форматов xlsx, csv, ods, pdf.
MOD // 03
Ячейки и диапазоны
Ввод, редактирование, копирование, объединение ячеек и работа с диапазонами.
MOD // 04
Формулы и вычисления
Разбор и вычисление формул. Функции SUM, AVERAGE, IF, VLOOKUP и автоматический пересчёт.
MOD // 05
Форматирование данных
Шрифты, цвета, границы, числовые форматы и условное форматирование ячеек.
MOD // 06
Диаграммы и графики
Построение гистограмм, линейных, круговых и точечных диаграмм на основе данных.
MOD // 07
Сортировка и фильтрация
Упорядочивание данных по столбцам и отбор строк по заданным условиям.
MOD // 08
Управление листами
Создание, переименование, копирование листов и межлистовые ссылки в формулах.
MOD // 09
Сводные таблицы
Группировка и агрегация данных с динамическим управлением полями и функциями.
MOD // 10
Проверка данных
Правила ввода: диапазоны чисел, списки значений, форматы дат и подсказки.
MOD // 11
Макросы и автоматизация
Запись и воспроизведение действий пользователя, назначение горячих клавиш.
MOD // 12
Совместная работа
Многопользовательское редактирование, комментарии и история изменений документа.
MOD // 13
Печать и экспорт
Настройки страницы, предварительный просмотр и экспорт в PDF.
MOD // 14
Статистика и анализ
Описательная статистика, регрессия и анализ «что если» с визуализацией результатов.
