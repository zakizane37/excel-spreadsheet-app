# Табличный процессор (аналог Microsoft Excel)

Данный репозиторий содержит имитацию разработки программного проекта «Табличный процессор».

## Цель проекта

Разработка программного продукта для работы с электронными таблицами, обеспечивающего ввод, хранение, обработку, визуализацию и экспорт табличных данных с поддержкой формул, диаграмм и инструментов анализа.

## Основные возможности

- Ввод и редактирование данных в ячейках
- Вычисление формул и стандартных функций
- Построение диаграмм и графиков
- Сортировка, фильтрация и анализ данных
- Условное форматирование и проверка данных
- Сводные таблицы и статистический анализ
- Совместная работа в режиме реального времени
- Экспорт в PDF, CSV, XLSX

## Структура проекта

Проект разделён на логические модули, каждый из которых отвечает за отдельную часть функциональности системы.

1. **Пользовательский интерфейс** — отображение рабочей области, лента инструментов, панели.
2. **Управление файлами** — создание, открытие, сохранение, импорт/экспорт файлов.
3. **Работа с ячейками и диапазонами** — ввод, редактирование, выделение, копирование.
4. **Формулы и вычисления** — разбор и вычисление формул, зависимости между ячейками.
5. **Форматирование данных** — шрифт, цвет, числовые форматы, условное форматирование.
6. **Диаграммы и графики** — построение и настройка графических представлений данных.
7. **Сортировка и фильтрация** — упорядочивание и отбор строк по условиям.
8. **Управление листами** — создание, переименование, копирование листов.
9. **Сводные таблицы** — группировка и агрегация больших массивов данных.
10. **Проверка данных** — правила ввода, ограничения и всплывающие подсказки.
11. **Макросы и автоматизация** — запись и воспроизведение действий, пользовательские скрипты.
12. **Совместная работа** — многопользовательское редактирование, комментарии, история.
13. **Печать и экспорт** — настройки страницы, предварительный просмотр, PDF.
14. **Статистика и анализ данных** — описательная статистика, регрессия, анализ «что если».
